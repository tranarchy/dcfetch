- OS_NAME
  - change this if you want to override the displayed OS name

- CPU_ARCH
  - if set to 1 it adds the CPU arch next to the OS name (e.g. Void Linux x86_64)

- OS_IMAGE
  - change this if you want to override the displayed OS image
  - values -> `void arch gentoo debian ubuntu fedora nixos macos default`

- ALT_IMAGE
  - if set to 1 it uses the alternative image for the OS

- OS_DETAILS
  - what to display below the OS name
  - values -> `DESKTOP SHELL TERM KERN`

- CUSTOM_DESC
  - changes this if you want a custom description instead of the OS details

- SEPARATOR
  - separator between the OS details

- DISPLAY_MEDIA
  - if set to 1 the currently playing media name is displayed

- FILTER_MEDIA_APPLICATIONS
  - whitelist applications for DISPLAY_MEDIA, (e.g. FILTER_MEDIA_APPLICATIONS="mpv firefox")
